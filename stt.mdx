---
title: "STT — Quickstart"
description: "Convert audio to text using Waves Lightning ASR. This endpoint supports raw audio bytes or public audio URLs, automatic language detection (30+ languages), and optional word-level timestamps."
---

## Endpoint

<Info>
  \<Endpoint

    method="POST"

    url="https://waves-api.smallest.ai/api/v1/lightning/get_text"

  /\>
</Info>

<Info>
  `POST ` /api/v1/lightning/get_text
</Info>

## Required Headers

<ParamField query="Authorization" path="param" body="body" header="Authorization" type="string<bearer>" required="false" placeholder="ph">
  API key authentication using Bearer token format.
</ParamField>

<ParamField query=" model" path="file" body="body" header="file" type="enum<string>" required placeholder="ph">
  The ASR model to use for transcription

  Available options: `lightning`
</ParamField>

<ParamField query="file" path="file" header="file" required>
  **Supported format:**\
  `Raw audio bytes` Content-Type header should specify the audio format (e.g., audio/wav, audio/mp3). All parameters are passed as query parameters.

  `url ` URL to the audio file to transcribe. Must be publicly accessible
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://waves-api.smallest.ai/api/v1/lightning/get_text \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/octet-stream'
```


```python Python expandable
import requests

url = "https://waves-api.smallest.ai/api/v1/lightning/get_text"

headers = {
    "Authorization": "Bearer <token>",
    "Content-Type": "application/octet-stream"
}

response = requests.post(url, headers=headers)

print(response.json())
```
